# **App Name**: Stundenzettel Meister

## Core Features:

- Benutzer Auswahl: Auswahl des Benutzers, dessen Name auf dem Stundenzettel erscheint.
- Arbeitszeiterfassung: Eingabe von Tag, Kundenname und Anzahl der Stunden in dezimaler Form.
- Stundenzettel-Ansicht: Anzeige der erfassten Arbeitszeiten im DIN A5 Format pro Tag.
- Monatsübersicht: Darstellung des kompletten Monats mit jeweils zwei DIN A5 Stundenzetteln auf einer DIN A4 Seite.
- PDF-Download: Herunterladen der Monatsübersicht als PDF-Datei.
- Liquid Glass Design: Integration des Liquid Glass React Design Pakets (https://github.com/rdev/liquid-glass-react) für eine ansprechende Benutzeroberfläche.

## Style Guidelines:

- Primary color: Soft blue (#A7C4BC), to echo a clean and professional feel, and also match glass-like ui themes well.
- Background color: Light gray (#F0F4F3), for a simple and non-intrusive look.
- Accent color: Pale cyan (#76C2AF) to accent selected days of the month in a manner that is easily visible, while also complementing the main design scheme.
- Headline font: 'Poppins', sans-serif, for headings and labels.
- Body font: 'PT Sans', sans-serif, for a very readable sans-serif.
- Anordnung von zwei DIN A5 Stundenzetteln pro DIN A4 Seite für eine optimale Druckansicht.
- Verwendung einfacher, intuitiver Icons zur Kennzeichnung verschiedener Aktionen (z.B. Speichern, Download).
